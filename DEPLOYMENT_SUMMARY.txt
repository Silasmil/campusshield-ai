KEY CONFIGURATION FILES
═══════════════════════════════════════════════════════════════════════════

To Deploy to Production, You Need:

1. .env.production (after copying from .env.production.example)
   Required variables:
   ├── DJANGO_ENVIRONMENT=production
   ├── DJANGO_SECRET_KEY=<60+ character key>         (Generated/provided)
   ├── ALLOWED_HOSTS=yourdomain.com                  (Your domain)
   ├── DATABASE_URL=postgresql://...                 (PostgreSQL connection)
   ├── DEBUG=False                                   (Security)
   ├── SECURE_SSL_REDIRECT=True                      (HTTPS)
   └── SESSION_COOKIE_SECURE=True                    (Cookie security)

2. SSL Certificate Files (for HTTPS)
   ├── /etc/letsencrypt/live/yourdomain/fullchain.pem
   └── /etc/letsencrypt/live/yourdomain/privkey.pem
   Generated via: certbot certonly --standalone -d yourdomain.com

3. PostgreSQL Database
   ├── Connection string in DATABASE_URL
   ├── Database created and accessible
   └── Initial migrations applied
